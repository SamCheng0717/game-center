底部快捷键的交互方式

点击快捷键切换"点击雷区的状态"——哪种理解是你想要的？

A

三模式切换（我的理解）

底部固定三个按钮，当前激活的模式决定点击格子的效果：

翻开
插旗
位置
当前模式：翻开 → 点格子 = 揭开
  • 翻开：点格子 → 揭开/排雷
  • 插旗：点格子 → 放置/取消旗帜
  • 位置：点格子 → 移动高亮光标，不触发任何操作（方便精确定位再切模式操作）
B

翻开+插旗+问号（Windows 三态）

复刻 Windows 右键三状态循环：

  • 翻开：点格子 → 揭开（左键效果）
  • 插旗：点格子 → 插旗标记（右键一次）
  • 位置：点格子 → 打问号"？"（右键两次，表示不确定）
□ → 🚩 → ❓ → □（循环）
C

两模式 + 确认操作

  • 翻开：激活后点格子 = 揭开
  • 插旗：激活后点格子 = 插旗
  • 位置：显示当前格坐标信息（行列数），帮助用户在手机小格子上精准定位
